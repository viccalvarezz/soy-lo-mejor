Lenguaje de programación: Java
Archivo de entrada: Ejercicio3y4/archivos/datos.xml
Archivo de salida: Ejercicio3y4/archivos/datosSalida.xml

Instrucciones:

  La aplicación utiliza una interfaz de usuario. Dentro de dicha interfaz se
encuentran las figuras geométricas que se crean en el xml de entrada, en este
caso 4, dos círculos y dos rectángulos sin relleno ni color alguno. En el
momento que se ejecuta la aplicación aparecen dibujadas las figuras geométricas
del xml introducido y además, dos botones, uno que permite colorear los círculos
y otro los rectángulos con colores predeterminados. Al finalizar, se genera el
archivo de salida datosSalida.xml añadiendole a los elementos figura el atributo
color y colocando el color correspondiente a la figura.

  Se quiso simular el comportamiento de aplicaciones reales que utilizan
interfaz gráfica usuario y archivos xml debido a que es el uso más común de los
xml.
